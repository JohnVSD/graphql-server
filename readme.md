## GraphQL 练习

启动项目 ```yarn dev``` 注意：该命令只是启动 src 目录下方的一级目录内容，包括db.js、resolvers.js、schema.graphql 三个文件；

src/graphql-js 文件下方是官方示例练习；使用node命令来启动。

#### 项目结构
```base
## src 文件结构

├─ db.js                    ## 模拟数据库文件
├─ resolvers.js             ## schema 数据处理器
├─ schema.graphql           ## schema 定义类型
├─ graphql-js               ## graphql 官方示例练习
|     ├─server01.js
|     ├─server02.js
|     ├─server03.js
|     ├─server04.js
|     ├─server05.js
|     └server06.js
```
